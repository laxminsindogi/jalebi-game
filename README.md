# 🍥 Jalebi Game

A colorful and addictive browser-based arcade game built using HTML5 Canvas, CSS, and JavaScript.  
Draw glowing spiral trails, collect sweets, level up, and avoid crossing your own path.

---

# ✨ Features

- 🎮 Smooth arcade-style gameplay
- 🌈 Neon glowing visual effects
- 🍬 Collectible sweet dots
- 💥 Particle burst animations
- 📈 Level progression system
- 🏆 Best score tracking
- 📱 Mobile & desktop support
- ⚡ Fast HTML5 Canvas rendering
- 🎨 Beautiful dark neon UI
- 🔥 Dynamic trail animations

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Canvas API
